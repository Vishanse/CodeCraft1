Learn HTML + CSS

CSS (Cascading Style Sheets) is used to style your HTML content with colors, fonts, layouts, and more.

  }
</style>

Learn JavaScript

JavaScript brings interactivity to your website like button clicks, image sliders, and more.

<script>
  function greet() {
    alert('Hello, JavaScript!');
  }
</script>

<button onclick="greet()">Click me</button>
